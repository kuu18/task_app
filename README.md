* タスク更新機能
* タスク削除機能
* タスク検索機能

* Java8
* HTML5
* MySQL
* git
* SpringBoot2.6.3
* SpringDataJpa
* SpringSecurity
* BootStrap5.1.3
* Thymeleaf
* VSCode
